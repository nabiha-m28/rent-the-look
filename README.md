# Rent the Look

A full-stack web app that converts retail product URLs into rental alternatives across Rent the Runway, Nuuly, and Pickle — with price comparisons, size filtering, and Pinterest-style boards for saving and organizing listings.

## Features

- Paste any product link and find it (or the closest match) across major rental platforms
- Retail vs. rental price comparisons with savings displayed
- Filter results by size
- Save listings to custom boards with Supabase-backed user accounts
